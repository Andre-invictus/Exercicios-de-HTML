Que significa semântica? Por que isso é tão importante em HTML5

semântica

Semântica é usa um código HTML e já entende o que cada parte da página faz, só pelas tags que estão sendo usadas, Isso é o que a gente chama de semântica no HTML5. Em vez de usar <div> pra tudo, a ideia é aproveitar elementos como <header>, <main>, <nav>, entre outros, que já dizem claramente o papel daquele trecho no site. Isso não só deixa o código mais limpo e fácil de manter, como também ajuda mecanismos de busca (tipo o Google) a entender melhor a estrutura da página — sem falar que facilita a vida de quem usa leitores de tela, como pessoas com deficiência visual. É aquele tipo de cuidado no desenvolvimento que faz toda a diferença.

Exemplos semântica

  • <p> - usado para criar paragrafo
  • <h1 ate h6> - Esse são utilizados para da niveis no titulo
  • <table> - Ele e utilizado para criar tabelas
  • <img> - Ele e utilizado para adicionar imagem no HTML
